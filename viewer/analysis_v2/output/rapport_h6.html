E01 — Topologie x Planeite (h6)

But : savoir si la composition en cycles (nombre de pentagones, nombre d'heptagones) predit la planeite de la molecule apres optimisation xTB.

Hypotheses chimiques : Les pentagones induisent une courbure positive (forme "bol"), les heptagones une courbure negative (forme "selle"). Quand on en combine, la courbure peut se compenser et donner du plan. On cherche les cellules ou la combinaison "marche".

Comment ca a ete calcule : Pour chaque cellule de la grille (n_pent, n_hept), on prend toutes les solutions correspondantes, on calcule la fraction de celles dont le verdict MD est 'plan'. On exige >= 5 sols pour qu'une cellule soit reportee.
3 cellules (n_pent, n_hept) avec >=5 sols. 0 cellules a >=90% plan (sur >=20 sols), 0 cellules a <=30% plan.

% solutions plan par (n_pent, n_hept)

Chaque case = une composition (n_pent x_axis, n_hept y_axis). La couleur indique le pourcentage de molecules planes apres MD.
Rouge fonce = ~100% plan (composition tres stable). Bleu fonce = ~0% plan (composition tordue par construction). Blanc ~50% plan (cas intermediaire). Les cellules vides = pas assez de donnees (n < 5).

log10(taille de l'echantillon) par cellule

Pour interpreter le heatmap precedent il faut savoir COMBIEN de solutions chaque cellule contient. Ce 2e heatmap montre cela en echelle log : valeur 2 = 100 sols, 3 = 1000 sols, etc.
Les cellules avec une couleur plus foncee (bleu fonce) sont les compositions LES PLUS FREQUENTES dans nos donnees. Une cellule avec %plan extreme mais peu de donnees doit etre regardee avec prudence (non significatif statistiquement).

E02 — Topologie x Aromaticite (h6)

But : comprendre quelles compositions (n_pent, n_hept) developpent le plus d'aromaticite (nombre de Clar eleve, RBO hex proche de 3).

Rappel :
- Clar number = nb max de sextets aromatiques places sur les hexagones de la molecule. Compris entre 0 (aucun rond) et le nb d'hex max (1 par hex possible).
- CBO hex moyen = aromaticite locale moyenne des hexagones. Valeur ideale 3 (benzene pur). NULL pour les molecules radicalaires (sans Kekule stricte).
- n_azulene_units = paires 5+7 fusionnees par arete (motif azulene, polaire, aromatique).

Hypothese : les compositions tres riches en hex avec quelques 5/7 bien places maximiseront l'aromaticite. Les compositions pent-pent ou hept-hept devraient avoir moins de Clar (et plus de radicaux).
Cellule la plus aromatique (Clar moyen max) : (1, 1) a 1.73 sextets en moyenne.
Cellule au CBO hex max : (2, 2) a 2.32/3 (3 = benzene).
Cellule avec le plus d'unites azulene : (3, 3) a 5.3.

Clar_number moyen par composition

Moyenne du nombre de Clar (max de sextets aromatiques places) sur les solutions plan de chaque cellule.
Rouge fonce = cellules tres aromatiques (plusieurs sextets en moyenne). Jaune pale = peu/pas de sextets. Les cellules avec beaucoup d'hex (haut-gauche de la grille) sont typiquement plus rouges.

CBO hex moyen par composition

Moyenne du RBO moyen sur les hexagones (aromaticite locale Pauling). Valeur 0 a 3 : plus c'est haut, plus les hex sont aromatiques.
Rouge fonce ~ 2.5-3.0 = hex tres aromatiques (comme benzene). Jaune pale = hex peu aromatiques (souvent dans les molecules radicalaires). NB : les molecules radicalaires sont exclues (CBO NULL).

n_azulene_units moyen par composition

Combien d'unites azulene (paire 5+7 partageant une arete) la molecule contient en moyenne. C'est un proxy de la polarite (cf litterature).
Bleu fonce = beaucoup d'unites azulene -> candidats pour la polarite/transfert de charge. Naturellement les cellules avec autant de 5 que de 7 (diagonale) en auront le plus.

E03 — Bordure x Stabilite (h6)

But : savoir si la forme de la bordure de la molecule (les atomes exterieurs) predit la planeite.

Concepts (issu de la these Varet, section 2.2) :
- Parametre d'irregularite (Bouwman 2019) = quantite chimique mesurant la "compacite" de la bordure. Calcule a partir des "groupes" de carbones consecutifs au bord. Valeur entre 0 et 1 : 0 = bordure tres reguliere/compacte (beaucoup de symetries), 1 = bordure tres irreguliere.
- n_pent_at_boundary = nombre de pentagones touchant le bord. Plus il y a de pent en bord, plus on s'attend a une courbure positive (bowl).

Hypothese : les bordures regulieres (irregularite proche de 0) devraient correspondre a des structures plus planes. Les bordures avec beaucoup de pent/hept en saillie devraient tordre la molecule.
Correlation irregularite x angle_deg (sur plan+non_plan) : -0.021 (QUASI-NULLE).
→ pas de lien clair.

% plan par bucket d'irregularite

On classe les solutions en 5 buckets selon leur irregularity_param, puis on affiche la fraction de chaque bucket qui est verdict='plan'.
Une barre haute = la majorite des sols dans ce bucket sont plans. Si la tendance est decroissante (vers la droite), les bordures irregulieres sont moins planes. Si plate, l'irregularite n'a pas d'effet observable.

% plan par nombre de pent au bord

Combien de % de solutions sont planes selon le nombre de pentagones qui touchent la bordure de la molecule.
Les pentagones induisent une courbure positive (forme bol). Si la barre baisse a mesure que n_pent_at_boundary monte, ca confirme l'effet "bowl" des pent en bord.

% plan par nombre de hept au bord

Idem mais avec les heptagones (courbure negative attendue).
Les heptagones induisent une courbure negative (forme selle). Si la barre baisse aussi, ca confirme l'effet "saddle" des hept en bord.

E04 — Geometrie 3D x Electronique (h6)

But : verifier si la courbure 3D de la molecule (mesuree apres optimisation xTB) influe sur son electronique (aromaticite, radicaux).

Concepts :
- curvature_discrete_mean = angle moyen entre les normales de cycles voisins. 0deg = plat parfait. ~30deg = courbure marquee (forme bol/selle).
- buckling_height = deviation max d'un atome au plan moyen (en Angstrom). 0 A = plat ; > 0.5 A = clairement courbe.
- Clar_number, cbo_mean_hex, n_radicals = indicateurs electroniques.

Hypothese (litterature, fullerenes) : plus une molecule est courbee, moins l'aromaticite est forte (les orbitales p ne se recouvrent plus aussi bien). On s'attend a une correlation negative entre courbure et Clar/CBO.
Pour les radicaux : pas d'hypothese claire dans la litterature, c'est exploratoire.
Pearson curvature x Clar (sur plans) : -0.091 (FAIBLE (pas de lien))
Pearson buckling x CBO hex (sur plans) : +0.110 (POSITIVE)
Pearson curvature x n_radicals (toutes sols) : -0.009 (FAIBLE (pas de lien))

Courbure 3D vs aromaticite (CBO hex)

Chaque point = une solution. X = courbure 3D moyenne (deg), Y = CBO hex moyen. Verts = plan, rouges = non_plan. Limite a 5000 points pour la lisibilite.
Si les points descendent vers la droite (forme \), ca confirme la litterature : courbure ↑ -> aromaticite ↓. Une separation visible verts/rouges = la courbure marque aussi la planeite (attendu). Un nuage diffus = pas de lien clair.

E05 — Localisation des radicaux (h6)

But : savoir ou tombent les electrons radicalaires des molecules non-Kekuleennes (n_radicals > 0). Question chimique posee par Liu & Feng (2020) : les radicaux sont-ils preferentiellement sur les pent, sur les hept, sur les hex, sur la bordure ?

Concepts :
- radical_on_pent_freq = sur les configurations radicalaires enumerees, fraction des radicaux tombant sur un atome de pentagone. Idem hex/hept/bordure.
- Pour les molecules NON-radicalaires (n_radicals = 0), ces champs sont NULL. On filtre donc uniquement les radicalaires.

Hypothese : les radicaux devraient preferentiellement tomber sur les sites de "stress topologique" : pentagones (electron en trop) ou bord. Les heptagones electronquerement peuvent aussi en attirer ("recovery of aromaticity" : creer un radical libere un tropylium aromatique a 6 electrons sur le 7-ring).

Note : un atome peut appartenir a plusieurs cycles, donc pent+hex+hept peut depasser 1 (chaque rad est compte dans tous ses cycles).
Statistiques sur 1318 solutions radicalaires. Frequence radicaux par site : pent 0.357, hex 0.486, hept 0.454, bordure 0.640.

Frequence radicaux par type de cycle hote

Moyenne de la frequence d'apparition d'un radical dans un cycle de chaque taille (pentagone, hexagone, heptagone). Calcule sur les solutions radicalaires uniquement.
La barre la plus haute = cycles qui "attirent" preferentiellement les radicaux. Si pent > hept >> hex, ca confirme l'hypothese que les radicaux se localisent sur les sites de stress topologique. Si hept eleve, c'est l'effet "tropylium aromatique" (recovery of aromaticity).

Repartition bord vs interieur

Fraction des radicaux qui tombent sur les atomes de bordure vs les atomes interieurs.
Si "bord" >> "interieur", les radicaux sont preferentiellement exposes (reactivite chimique forte). C'est le cas attendu pour les PAH avec radicaux stables.

Distribution n_radicals (TOUTES solutions)

Combien de solutions ont 0 radical, 1 radical, 2 radicaux, etc. Pas filtree par verdict.
Le pic est typiquement a 0 (la plupart sont Kekule strictes) ou 1 (parite forcee par les 5/7). Une queue a 2-3 = molecules a forte instabilite (candidats interessants pour magnetisme moleculaire).

E07 — Scaling h6 -> h9

But : comparer les quatre tailles (h6, h7, h8, h9) sur tous les indicateurs pour voir comment ils evoluent avec la complexite croissante.

Questions chimiques sous-jacentes :
- Le % de structures planes baisse-t-il avec la taille ? (plus grand = plus de contraintes geometriques)
- Le Clar moyen augmente-t-il ? (plus d'hex disponibles pour des sextets)
- Le n_radicals moyen augmente-t-il ou diminue-t-il ?
- Le buckling moyen (deviation 3D) augmente-t-il ? (les grosses molecules plient plus)

Note : cette section utilise l'ensemble des donnees, peu importe le filtre h du rapport (pour permettre la comparaison cross-h).

Fraction de solutions verdict='plan' a chaque taille.

Nombre moyen de sextets aromatiques places. Compte 0 quand pas d'hex.
